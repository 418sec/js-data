Class: Container

Container

new Container(opts)

The Container class is a place to define and store Mapper instances.

Container makes it easy to manage your Mappers. Without a container, you need to manage Mappers yourself, including resolving circular dependencies among relations. All Mappers in a container share the same adapters, so you don't have to register adapters for every single Mapper.
